Saving America

Role: Full Stack Development

Work done under Bixly
Site

Saving America is a site based around fundraising campaigns. It's sort of like gofundme combined with people management. I worked on many aspects of the site ranging from implementing mockups of the marketing/campaign/event pages to working on the front end and back end of event pages.

It was a challenging endeavor as I would have to implement interactive UIs for managing volunteers as well as dealing with some of the back end code associated with it.

Magpie Map

Role: Front End Development

Work done under Bixly
Site

This map was created for Magpie Health Analytics who wanted to prove that SAS Health Analytic jobs exist in the U.S. The solution? A responsive Leaflet map where I implemented custom data aggregation to show admin inputted jobs.

The map automatically zooms in depending on user location and also allows the user to favorite jobs in their browser's local storage for later retrieval (this feature was removed for the sake of simplicity by the client).

Run For People is an ecommerce store built on Python/Django. I built the site from the ground up and it allowed users to register, buy medals, participate in virtual races, and then proceed to posting results of their runs.
