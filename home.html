Vocalize

React/Redux, NodeJS/Express, MongoDB

Github repository

Vocalize is an app where users can upload audio clips and listen/vote on other audio clips. The idea is to semianonymously answer the question of can you sing?

The app is currently under heavy development and what I'm currently spending most my free time on. It uses a React/Redux client and Node/Express API.

Github repository

Photo sharing social media site built with Python/Django. Based on Instagram's design.
